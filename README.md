# 🔐 Sistema de Autenticação Web - PHP

Este projeto demonstra a implementação de uma interface de login funcional integrada ao backend, focada em segurança e captura de dados simples.

## 🛠️ Tecnologias Utilizadas
* **HTML5/CSS3**: Interface moderna baseada em componentes de redes sociais.
* **PHP**: Processamento de dados no lado do servidor.
* **Segurança**: Implementação de `htmlspecialchars` para evitar vulnerabilidades de XSS e uso de método `POST`.

## 🚀 Funcionalidades
- **Captura de Dados**: O sistema recebe o usuário via formulário.
- **Feedback Dinâmico**: Exibe uma mensagem de confirmação processada pelo PHP.
- **Design Responsivo**: Layout que se adapta a diferentes tamanhos de tela.

## 💻 Como testar
1. Você vai precisar de um servidor local (como o **XAMPP**).
2. Coloque o arquivo `login.html` (renomeie para `login.php` se for rodar o script) na pasta `htdocs`.
3. Acesse `localhost/seu-arquivo` no navegador.
